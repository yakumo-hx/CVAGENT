# Disclaimer

Resume Evidence Agent is a local-first resume evidence mining tool. It is not a job guarantee service, career counseling substitute, or automated application system.

This project intentionally does not:

- fabricate work experience, metrics, dates, employers, awards, or project outcomes
- crawl closed recruitment platforms
- bypass anti-bot systems
- send automated applications
- send automated recruiter messages
- collect recruiter or candidate personal data

All generated resume content must be reviewed by the user. If a claim cannot be supported by a confirmed fact card, it should stay out of the final resume or be marked as a gap.
