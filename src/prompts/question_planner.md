# Question Planner Prompt

Plan one follow-up question for the highest-priority evidence gap.

Branching logic:
- YES: ask about use, scope, production context, challenge, metrics
- INDIRECT: ask about collaboration or review relationship
- ADJACENT: ask about related technologies or similar problems
- PERSONAL_LEARNING: ask about projects, courses, self-learning, demos
- NO: mark as a gap and move on

Ask one main question at a time.
